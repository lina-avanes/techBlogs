Installing Apache on an AWS EC2 instance via Ansible

Installing Apache via Ansible on an AWS EC2 instance may seem like a complicated task; however, it is not. In todayâ€™s blog, I will demonstrate the steps needed in order to complete this task.

Prerequisites:

  • A running AWS EC2





Step 1: Create the Ansible Files


In order to install apache, you will need two files. The first file will keep the IP address(s) of the EC2 instance. The file is called "hosts.ini". The file will contain the command [Web_server] and the IP addresses(s).

An example of hosts.ini file:

[Web_server]
192.168.1.1


The second file will contain all the commands that will be executed on the EC2 instance.
