Installing Apache on an AWS EC2 instance via Ansible

Installing Apache via Ansible on an AWS EC2 instance may seem like a complicated task; however, it is not. In todayâ€™s blog, I will demonstrate the steps needed in order to complete this task.

Prerequisites:

  • A running AWS EC2





Step 1: Create the hosts file


In order to install apache, we will need two files. The first file will keep the IP address(s) of the EC2 instance. In another words, we wil be creating an Ansible inventory file. The inventory file called "hosts.ini" will define the hosts upon which tasks and commands in a playbook operate. The inventory file can have an individual hosts or groups of hosts. In this example, I have included a single host.

An example of hosts.ini file:

[Web_server]
172.16.8.1





Step 2: Create the ansible playbook file


The second file "tasks.yml" will contain all the commands/tasks that will be executed on the EC2 instance.

---
- hosts: web_server
  tasks:
    - name: Install required packages
      apt:
        name:
          - apache2
        state: present
      become: true
